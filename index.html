Community reviewed smart contracts.

Code 423n4 is an experimental open organization determined to build a new method of reviewing smart contract security. C4 encourages aggressive code analysis using open bug bounties where participants eat what they kill and compete to become champions of the arena.

There are three actors in Code 423n4:

WARDENS
Wardens hunt for bugs in the smart contracts of decentralized protocols.

SPONSORS
Sponsors create bounty pools to attract wardens to their contests.

JUDGES
Judges are responsible for allocating bounty pools to wardens based on their performance.
